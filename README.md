# tete


すきな人とのまいにちを、ちょっとずつためていける場所だよ。
デートの思い出も、好きな食べ物も、ぜんぶ「ふたりのノート」にきゅっと。

---

## なにができるの？

-  **記念日カウンター** — 付き合った日からの日数をかぞえて、ずっと見守る
-  **デートの記録** — どこ行った・なに着た（かぶり防止！）をのこせる
-  **ふたりのバケツリスト** — 行きたい場所ややりたいこと、叶えたらチェック✔️
-  **スキンケアログ** — 使ったコスメと肌の調子を記録して、自分だけのきれいを育てる
-  **好み図鑑 & 地雷メモ** — 好き・きらい・アレルギー…相手を「ちゃんと知る」ノート
-  **プレゼント記録** — あげた・もらったをこっそりメモ（サプライズはナイショね）
-  **SNSカード生成** — 思い出をかわいいカードにして、シェアできちゃう

---

## こんな人に

すきな人との毎日を、もっとていねいに残したいあなたへ。
`#tete` でみんなにも自慢しちゃお
